# serviceregistrator

An alternative to https://github.com/gliderlabs/registrator

## virtual env

### Create

```bash
python3 -m venv .venv
```

### Activate

```bash
source .venv/bin/activate
```

## Dependencies

```bash
pip install -r requirements/base.txt
```

## Running (dev)


```bash
pip install -e .
```

```bash
serviceregistrator
```
